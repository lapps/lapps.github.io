Questions/issues:

  1. Similar to the coreference structure, should the children be local identifiers?
  2. The contains list would get rather long since it has as many elements as there are categories in the parse. Actually, I think I know the answer. The contains should probably contain something like categorySet:penn. We have not settked yet on the defiitive way to encode this as discriminators.
