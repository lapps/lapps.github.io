Phrase Structure in LIF

Last updated: October 29th, 2015

We use two annnotation object from the vocabulary, PhraseStructure and Consituent:

  • http://vocab.lappsgrid.org/PhraseStructure.html
  • http://vocab.lappsgrid.org/Constituent.html

The PhraseStrucutre object contains a single parse tree for some text span, typically for a sentence. It introduces a feature named constituents which contains a list of identifiers referring to (1) annotations of @type Constituent in the same view or (2) annotations of @type Token that may live in another view. A Constituent has a label reflecting the category and in the features dictionary a list of children.

{
  "text": "Sue sees herself",
  "views": [
    { "id": "v1",
      "metadata": {
        "contains": {
          "Token": {
            "producer": "edu.brandeis.cs.lappsgrid.opennlp.Tokenizer:n.n.n",
            "type": "tokenizer:opennlp" }}},
      "annotations": [
         { "@type": "Token", "id": "tok0", "start": 0, "end": 3 },
         { "@type": "Token", "id": "tok1", "start": 4, "end": 8 },
         { "@type": "Token", "id": "tok2", "start": 9, "end": 16 } ]},
    { "id": "v2",
      "metadata": {
        "contains": {
          "PhraseStructure": {
            "producer": "edu.brandeis.cs.lappsgrid.SimpleParser:1.0.0",
            "categorySet": "ns/types/PTBcategories",
            "type": "PhraseStructure:SimpleParser" },
          "Constituent": {
            "producer": "edu.brandeis.cs.lappsgrid.SimpleParser:1.0.0",
            "categorySet": "ns/types/PTBcategories",
            "type": "PhraseStructure:SimpleParser" }}},
      "annotations": [
         { "@type": "PhraseStructure",
           "id": "phrase0",
           "start": 0,
           "end": 16,
           "features": {
              "constituents": [ "c0", "c1", "c2", "v1:tok0", "v1:tok1", "v1:tok2" ] }},
         { "@type": "Constituent",
           "label": "S",
           "id": "c0",
           "features": {
             "children": [ "c1", "c2"] } },
         { "@type": "Constituent",
           "label": "NP",
           "id": "c1",
           "features": {
             "children": [ "v1:tok0" ] }},
         { "@type": "Constituent",
           "label": "VP",
           "id": "c2",
           "features": {
             "children": [ "v1:tok1", "v1:tok2" ] }}]}]
}
